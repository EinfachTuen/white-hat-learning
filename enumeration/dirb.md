## Dirb
Directory brute-forcing tool
- super fast
- default wordlist, can be changed
- can specify extensions
- can specify number of threads

### Usage
```sh
dirb http://<ip>
dirb http://<ip>:<port> 
dirb http://<ip>:<port> /usr/share/wordlists/dirb/common.txt
```
